# AI-Powered Personal Finance Analyzer

## üìã Overview
The **AI-Powered Personal Finance Analyzer** is a tool designed to analyze and generate detailed financial insights from **UPI expenses PDFs**. Using **Gemini AI**, the app processes the transaction history, generates structured insights, and offers actionable suggestions to help users manage their finances effectively.

### Key Features:
- **Monthly Income & Expenses** breakdown
- **Unnecessary Expense Categories** identification
- **Estimated Savings %** calculation
- **Spending Trends** analysis
- **Category-wise Expense Breakdown (Partial)** 
- **Cost Control Suggestions** for improved budgeting
- **AI-generated financial report** downloadable

## üöÄ How to Use

### Step 1: Upload your **UPI Expenses PDF**
Upload your UPI transaction history in PDF format using the "Upload PDF" button.

### Step 2: Analyze the Data
Click the "Analyze" button to trigger the AI-powered analysis. The app will process your PDF and generate a structured financial report.

### Step 3: View Insights
The generated insights will appear on the UI, including:
- Monthly Income & Expenses
- Unnecessary Expense Categories
- Estimated Savings %
- Spending Trends
- Category-wise Expense Breakdown
- Cost Control Suggestions

### Step 4: Download AI Report
Once the analysis is complete, you can download a detailed financial report containing all the insights generated by the AI model.

## üß† How It Works

1. **Text Extraction**: The PDF is processed using the `PyPDF2` library to extract text from each page.
2. **Data Analysis**: The extracted transaction history is sent to the **Gemini AI model**, which processes the data and generates detailed financial insights.
3. **Output**: The app displays the insights on the frontend, and a downloadable **.txt** file containing the full report is provided for further analysis.

## üîë Gemini API Configuration

To use the Gemini AI model, you need to configure the Gemini API key in the script.

1. Obtain your Gemini API key from Google.
2. Set the key in the `GEMINI_API_KEY` variable in the script:

```python
GEMINI_API_KEY = "Your_Api_Key"
```
